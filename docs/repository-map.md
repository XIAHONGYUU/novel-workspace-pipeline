# Repository Map

## Top Level

- `README.md`
  Product-facing overview.
- `WORKFLOW.md`
  High-level operating model.
- `CONTRIBUTING.md`
  Contribution rules.
- `CURRENT_STATUS.md`
  Optional repo-level pointer file.

## Workflow Layers

- `novel-chapter-distillation-skill/`
  Chapter skeleton and stage-shift scaffolding plus validation.
- `novel-opening-analysis-skill/`
  First-three-chapter analysis scaffolding plus validation.
- `novel-protagonist-encyclopedia-skill/`
  Protagonist-centered workspace bootstrap and validation.
- `novel-outline-analysis-skill/`
  Whole-book structural analysis scaffolding plus validation.
- `novel-highlight-scenes-analysis-skill/`
  High-impact scene analysis scaffolding plus validation.

## Orchestration

- `novel-workspace-orchestrator-skill/`
  Status collection, next-step recommendation, execution routing, gap reports, repair plans, and regression harness.

## Helper Tools

- `text2markdown/`
  Local `.txt` to `.md` conversion helper.
- `novel-character-cards/`
  Character extraction pipeline used by protagonist bootstrap flows.

## Documentation

- `docs/`
  Product-facing supporting documentation and example configuration files.
