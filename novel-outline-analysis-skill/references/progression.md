# 升级流结构参考

适用于：

- 修仙 / 玄幻 / 巫师 / 系统 / 无限升级类小说
- 主角成长阶梯非常清晰的小说

## 优先看的结构槽位

- 外挂机制或核心手艺
- 等级 / 境界 / 升级路线
- 关键质变节点
- 资源线与突破线
- 高位路线与终局位格

## 阶段划分时优先找

- 第一次正式入门
- 第一次大质变
- 普通成长线失效的时刻
- 高位结构真正展开的时刻
- 终局问题第一次成形的时刻

## 常见误判

- 把每次升级都当一个独立阶段
- 只盯着境界名，不看真正的结构换挡
- 忽略“资源模式变了”其实就是阶段边界

## 单书特性候选示例

- 外挂机制
- 升级路线
- 核心资源链
- 形态质变
- 高位结构抬升
