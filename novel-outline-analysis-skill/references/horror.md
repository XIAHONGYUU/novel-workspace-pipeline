# 恐怖民俗流结构参考

适用于：

- 怪谈 / 民俗 / 规则 / 仪式 / 禁忌 / 诡异生存类小说

## 优先看的结构槽位

- 恐怖规则来源
- 禁忌或仪式系统
- 真相揭示层级
- 生存压力如何升级
- 恐怖与现实线如何咬合

## 阶段划分时优先找

- 第一次真正见鬼 / 见规则
- 主角第一次掌握对抗手段
- 恐怖从局部事件升级为系统性灾难
- 真相从民间层升到宗教 / 历史 / 世界层
- 终局是镇压、共存、循环，还是更高层重解释

## 常见误判

- 只记录恐怖事件，不判断规则升级
- 把每个副本都当主线
- 忽略现实求生线和超自然线的交叉关系

## 单书特性候选示例

- 规则恐怖系统
- 仪式与禁忌
- 真相揭示阶梯
- 恐怖升级链
- 人道 / 现实线回环
