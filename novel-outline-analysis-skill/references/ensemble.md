# 群像关系流结构参考

适用于：

- 关键关系线、配角压力、群像互动、反派功能分层占很大比重的小说

## 优先看的结构槽位

- 核心关系回环
- 关键配角的功能分层
- 反派的结构功能
- 团队 / 阵营内部张力
- 情绪回响如何支撑高潮与结尾

## 阶段划分时优先找

- 关系格局第一次成形
- 关键关系破裂 / 重组
- 配角从辅助角色升级成结构支点
- 反派功能从局部阻碍升级为终局压力
- 结尾是否回收关键关系线

## 常见误判

- 把所有配角都写成“重要”
- 只写人设，不写剧情功能
- 只写反派强弱，不写反派在结构里的作用

## 单书特性候选示例

- 关键关系回环
- 群像压力结构
- 反派功能分层
- 情绪回响支撑终局
