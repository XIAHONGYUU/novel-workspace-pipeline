# 阵营政治流结构参考

适用于：

- 王朝 / 官场 / 帝国 / 军政 / 门阀 / 学派 / 大组织博弈类小说

## 优先看的结构槽位

- 阵营格局
- 制度变化
- 权力重分配
- 关键联盟与背叛
- 战争 / 政变 / 改制节点

## 阶段划分时优先找

- 主角第一次真正入局
- 主角第一次获得制度性位置
- 地方冲突升级为国家级冲突
- 单点事件升级为阵营再分配
- 终局秩序如何收束

## 常见误判

- 把所有势力都平铺，不区分主次
- 只写谁打谁，不写制度层变化
- 忽略“换舞台”其实意味着换权力规则

## 单书特性候选示例

- 阵营博弈
- 制度换挡
- 权力阶梯
- 战争抬升
- 终局秩序重写
